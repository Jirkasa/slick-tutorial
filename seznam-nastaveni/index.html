Seznam nastavení

Na této stránce se nachází seznam všech nastavení, které nám Slick poskytuje.

Šipky

arrows - zapíná/vypíná zobrazení šipek

appendArrows - mění ve kterém elementu se mají zobrazit šipky

prevArrow - nastavuje který element bude představovat šipku zpět

nextArrow - nastavuje který element bude představovat šipku dopředu

Navigace

dots - zapíná/vypíná zobrazení navigace

appendDots - mění ve kterém elementu se má zobrazit navigace

dotsClass - mění jméno třídy pro navigaci slideru

Slidy

initialSlide - nastavuje startovní slide

slidesToShow - nastavuje kolik se ve slideru najednou zobrazí slidu

variableWidth - nastavuje že se nebude měnit šířka slidů ale použije se jejich vlastní

centerMode - zapíná/vypíná vycentrování aktivního slidu

centerPadding - nastavuje mezery po stranách slideru při zapnutém center módu

lazyLoad - zapíná/vypíná lazy loading obrázků

zIndex - nastavuje CSS vlastnost z-index všem slidům ve slideru

Posouvání slidů

fade - zapíná/vypíná fade efekt, který se použije namísto posouvání slidů

infinite - zapíná/vypíná nekonečnost slideru

autoplay - zapíná/vypíná automatické přehrávání

autoplaySpeed - nastavuje rychlost automatického přehrávání

pauseOnHover - zapíná/vypíná pozastavení automatického přehrávání při najetí myši na slider

pauseOnDotsHover - zapíná/vypíná pozastavení automatického přehrávání při najetí myši na navigaci slideru

pauseOnFocus - zapíná/vypíná pozastavení automatického přehrávání při focusnutí slideru

accessibility - zapíná/vypíná možnost posouvání slidů pomocí šipek při focusnutí slideru

slidesToScroll - nastavuje o kolik slidů se bude ve slideru posouvat

draggable - zapíná/vypíná možnost posouvání slidů pomocí myši

swipe - podobné jako nastavení draggable, nevím v čem je rozdíl

swipeToSlide - zapíná/vypíná posunutí na slide na který jsme se myší posunuli nejblíže

edgeFriction - nastavuje velikost odporu krajních stran u konečného slideru

touchMove - zapíná/vypíná posouvání slidů při pohybu myši

touchThreshold - nastavuje vzdálenost, kterou musíme při posouvání slidů myší přejet abychom se dostali na další slide

vertical - zapíná/vypíná vertikální mód

verticalSwiping - zapíná/vypíná posouvání slidů myší vertikálně namísto horizontálně

rtl - nastavuje směr slideru na right-to-left

speed - nastavuje rychlost posouvání/prolínání slidů

waitForAnimate - povoluje/zakazuje interakci se sliderem při posouvání/prolínání slidů

useCSS - zapíná/vypíná používání CSS transition vlastností

useTransform - zapíná/vypíná používání CSS transform vlastností

cssEase - nastavuje CSS ease funkci pro posouvání/prolínání slidů

easing - nastavuje JQuery easing metodu

Responzivita

responsive - umožňuje měnit nastavení slideru podle šířky obrazovky

mobileFirst - zapíná/vypíná mobile-first strategii pro responzivitu

respondTo - mění podle čeho se bude brát šířka pro responzivitu

Ostatní

asNavFor - nastavuje slider jako navigaci pro jiný slider

rows - nastavuje kolik bude mít slider řad

slidesPerRow - nastavuje kolik slidů bude obsahovat každý řádek slideru

focusOnSelect

slide

customPaging
